# Defenestra, LLC

**Freedom from Windows®. Without the Compromise.**

Defenestra (/dɛfɛˈnɛstrə/) builds tools that help people and organizations
switch to Linux® without leaving their apps, files, or workflows behind.

## We're mostly on GitLab

Our source code lives at **[gitlab.com/defenestra-software](https://gitlab.com/defenestra-software)**.
This GitHub organization hosts **defenestraOS** only, everything else (Hyperpane,
Chassis, Slipgate, Medbay, and the rest of our suite) is on GitLab.

## What We Build

A suite of tools for the Linux desktop:

- **Hyperpane** - Run Windows apps on Linux without dual booting. Automated VM
  setup with graphics passthrough for near-native speed, telemetry blocking,
  and seamless file sharing.
- **Chassis** - A real device manager for Linux. See your hardware, manage
  drivers, monitor drive health, and inspect IOMMU groups without the terminal.
- **Slipgate** - Move your files, apps, and settings from Windows®, macOS®,
  or Linux® to a new Linux machine. Installs Linux alternatives for 130+
  Windows apps automatically.
- **Medbay** - Fix Linux problems without being a Linux expert. Tested,
  team-written fixes you can browse or describe in plain language. Runs
  entirely on-device.
- **defenestraOS** - A universal-blue-based Linux distribution. *Hosted here on GitHub.*

## Why Linux?

Private by default. Free to use. Runs well on older hardware. Updates on
your terms - not in the middle of your workday.
